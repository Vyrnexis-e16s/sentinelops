# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

Be kind. Assume good faith. Disagree with ideas, not people. Don't harass anyone for any reason. If someone asks you to stop, stop.

## Reporting

If someone makes this space feel unsafe, email m11k.08k3@gmail.com. Reports stay between you and the maintainer unless you say otherwise.
