# Notebooks

EDA + model interpretability work happens here.

- `eda.ipynb` — feature distributions, correlation, target balance, PCA projection.
- `feature_importance.ipynb` — RandomForest gini importances + permutation importance comparison.
- `errors.ipynb` — confusion matrix walkthrough, per-class precision/recall, lift curves.

These are intentionally not committed to CI — open them locally with JupyterLab.
